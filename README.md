# SocketDrop

SocketDrop is a lightweight real-time file sharing app built with Java and Spring Boot.  
Users connect via WebSockets, join rooms, and share files with live progress updates â€” without authentication or a database.

## Features
- WebSocket-based real-time communication
- Room-based file sharing
- File upload & download via HTTP
- Live transfer progress updates
- QR-code based room join (URL)
- In-memory storage only

## Tech Stack
- Java 17
- Spring Boot
- Spring WebSocket
- Spring Web (REST)
- Maven

## Notes
- No authentication
- No database
- Designed for learning and demonstration purposes
